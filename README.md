# Website Template Showcase

A ready-to-deploy React/Vite sales site for showcasing three website packages:

- Beginner
- Intermediate
- Pro

## Run locally

```bash
npm install
npm run dev
```

## Railway

Push to GitHub and connect the repo in Railway.

```txt
Build Command: npm run build
Start Command: npm start
```

The `railway.toml` is already included.
